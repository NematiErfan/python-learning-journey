# Project 2 - Python Comparison Operators and Statements

## Description

This step covers Python Comparison Operators and Python Statements.

The projects in this section apply the concepts learned from both topics in practical command-line applications.

## Topics Covered

   * Dictionaries
   * Variables
   * User Input
   * Comparison Operators
   * `if`, `elif`, and `else` Statements
   * `for` Loops
   * `while` Loops
   * `in` Operator
   * `del` Statement
   * `int()` and `float()`
   * Arithmetic Operations

## Projects

### 1. Student Grade Management System

A command-line application for managing student names and grades.

Features include:

   * Show students and their grades
   * Calculate the average grade
   * Find the highest grade
   * Find the lowest grade
   * Add a new student
   * Remove a student
   * Search for a student
   * Update a student's grade
   * Show students with grades above average
   * Display grade statistics

### 2. Guess Number Game

A command-line number guessing game where the player tries to guess a randomly generated number between 1 and 100.

The game provides feedback based on the player's guesses, such as:

   * `WARM`
   * `COLD`
   * `WARMER`
   * `COLDER`

## Files

   * `student_grade_manager.py` — Student Grade Management System.
   * `guess_number_game.py` — Guess Number Game.
